|$O(N^2)$|$O(1)$|

- 두 개씩 비교하면서 배열의 가장 큰 값을 넘기면서 끝에 몰아넣는다.
- 이제 끝에 몰아넣은 수를 제외하고 다음 번으로 큰 수를 찾으며 넘긴다.

## 코드
```java
public class Main {
    public static void bubbleSort(int[] arr) {
        for (int i = 1; i < arr.length - 1; i++) {
            for (int j = 0; j < arr.length - i; j++) {
                if (arr[j] > arr[j + 1]) {
                    int tmp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = tmp;
                }
            }
        }
    }
```
